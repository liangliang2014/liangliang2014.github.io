我学习了使用 VSCode 编辑器。

操作 快捷键
新建文件 ⌘ + n
保存 ⌘ + s
撤销 ⌘ + z
反撤销 ⇧ + ⌘ + z
格式化 ⇧ + ⌥ + f
注释 ⌘ + /
打开内置命令行 ⌃ + `
复制 ⌘ + c
剪切 ⌃ + x
粘贴 ⌃ + v
运行（调试） F5
自动补全 Enter
取消补全 Esc

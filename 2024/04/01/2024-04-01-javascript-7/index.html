省略花括号

if、for、while 这些控制结构，当只有 一行语句 时，可以省略花括号（但是不推荐）

if 省略花括号

  • 单独占一行
  • 跟 if、else 放在一行

for 省略花括号

  • 单独占一行
  • 跟 for 放在一行

while 省略花括号

  • 单独占一行
  • 跟 while 放在一行

真假值

  • 假值：0、''、NaN、null、false、undefined
  • 真值：其他都为真值
情况 相等性
NaN 连自身都不 ==（NaN != NaN）
null、undefined 这两个互相 ==（null == undefined）
0、''、false 这三个互相 ==（0 == '' == false）

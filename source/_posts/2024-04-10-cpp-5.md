### 函数传参

函数体中，形参的改变，不影响实参。

```cpp
#include <iostream>
using namespace std;

int f(int x) {
  x = 100; // 改变形参，不会影响到实参
  return x*2;
}

int main(int argc, const char * argv[]) {    
    int a = 10;
    int b = f(a);       // 实参 a 的值为 10

    cout << a << endl;  // 函数调用完后，实参 a 的值还是 10    
    return 0;
}
```

### 函数传指针

通过传指针，可以让函数通过指针改变实参的值。

```cpp
#include <iostream>
using namespace std;

void f(int* p) {
  *p = 100; // 没有改变形参，但是改变了形参指向的值
}

int main(int argc, const char * argv[]) {    
    int a = 10;
    f(&a);              // 传入 a 的地址 &a

    cout << a << endl;  // a 的值被改变了，100    
    return 0;
}
```

### 示例：用函数交换变量

```cpp
#include <iostream>
using namespace std;

void swap(int* p, int* q) {
  int t = *p;
  *p = *q;
  *q = t;
}

int main(int argc, const char * argv[]) {    
    int a,b; cin>>a>>b;
    swap(&a, &b);

    cout <<a<<", "<<b<<endl;
    return 0;
}
```
